# recruitment transisi

Aplikasi ini dibuat untuk memenuhi persyaratan berupa test yang diberikan oleh PT. Transisi Teknologi Mandiri
Semoga dapat memenuhi kriteria penilaian yang ditentukan.

Catatan :
1. Aplikasi dibuat dengan Flutter framework dan memanfaatkan GetX architecture dan API reqres.in
2. Semua fungsi yang berkaitan dengan API ada dalam folder controller/controller.dart
3. Aplikasi dalam bentuk APK diberi nama test.apk
4. Ada fitur session dengan SharedPreference untuk menyimpan token
5. Login menggunakan data dari reqres.in, yaitu :
